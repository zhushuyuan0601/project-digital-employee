## 四、技术方案

### 4.1 技术架构

```
前端（H5）
  ├─ Vue 3
  ├─ Vant UI
  └─ Axios
  
后端 API
  ├─ Node.js / Python
  ├─ Express / FastAPI
  └─ OpenAI SDK
  
大模型
  ├─ 鸿鹄大模型
  ├─ OpenAI 兼容接口
  └─ 模型名：honghu
  
数据存储
  ├─ Redis（会话）
  ├─ MySQL（用户数据）
  └─ ES（知识库）
```

### 4.2 API 设计

| API | 方法 | 功能 |
|-----|------|------|
| `/api/chat` | POST | 对话接口 |
| `/api/user/package` | GET | 套餐查询 |
| `/api/user/balance` | GET | 余额查询 |
| `/api/order/create` | POST | 业务办理 |
| `/api/ticket/create` | POST | 工单提交 |
