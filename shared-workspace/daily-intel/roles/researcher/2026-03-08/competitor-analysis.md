## 核心功能分析

### 最热门的产品能力

#### 1. AI视频生成（核心功能）

| 功能 | 描述 | 亮点 |
|------|------|------|
| **文生视频** | 一句话生成国风拜年视频 | 零门槛、秒级生成 |
| **图生视频** | 上传照片生成动态视频 | 全家Q版动画、萌宠拜年 |
| **AI做同款** | 半定制方案，修改提示词 | 省心省力、创意十足 |
| **AI全家福** | 一键焕新新年全家福 | 场景化应用 |

#### 2. 一键设铃（核心差异化能力）⭐

| 项目 | 说明 |
|------|------|
| **功能描述** | AI生成视频后一键设为彩铃 |
| **核心优势** | 10秒内完成，即刻生效 |
| **差异化价值** | 其他平台只能转发社交平台，无法嵌入通信环节 |
| **用户价值** | 解决传统彩铃制作繁琐、门槛高的问题 |

#### 3. 新年专属接线员

| 项目 | 说明 |
|------|------|
| **功能描述** | 上传照片、选方言音色，生成AI拜年语音助手 |
| **核心能力** | 未接电话时自动送上祝福 |
| **方言支持** | 支持多种方言音色（含安徽方言等） |
| **个性化** | 可设置状态、自定义祝福语 |

#### 4. AI特效模板

| 模板类型 | 描述 |
|----------|------|
| **节日特效** | 春节、中秋等节日模板 |
| **场景切换** | 同一句祝福、不同种表达 |
| **国潮风格** | 舞狮、灯笼、烟花等国风元素 |
| **企业模板** | 行业模板、品牌定制 |

#### 5. 星小辰智能体

| 项目 | 说明 |
|------|------|
| **功能定位** | 内置AI智能助手 |
| **核心能力** | 多轮交互与意图分析 |
| **使用方式** | 对话式创作，替代复杂操作 |
| **能力覆盖** | 文生图、图生视频、文生音乐等 |
