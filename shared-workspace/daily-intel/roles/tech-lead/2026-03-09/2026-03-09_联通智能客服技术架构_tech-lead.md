## 【鸿鹄大模型集成】

### API配置

```javascript
const HONGHU_CONFIG = {
  baseURL: 'https://api.honghu.com/v1',
  apiKey: 'your-api-key',
  model: 'honghu'
}
```

### 请求格式（OpenAI兼容）

```javascript
// 请求
POST /v1/chat/completions
{
  "model": "honghu",
  "messages": [
    { "role": "system", "content": "系统提示词" },
    { "role": "user", "content": "用户问题" }
  ]
}

// 响应
{
  "choices": [{
    "message": {
      "role": "assistant",
      "content": "AI回复"
    }
  }]
}
```
