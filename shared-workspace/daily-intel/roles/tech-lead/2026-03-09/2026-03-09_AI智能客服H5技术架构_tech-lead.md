# AI智能客服 H5 - 技术架构

> **日期**: 2026-03-09
> **技术负责**: 小开

## 【技术架构】

```
┌─────────────────────────────────────┐
│           Vue 3 + Vant UI           │
│  ┌────────┐ ┌────────┐ ┌────────┐  │
│  │对话界面│ │历史记录│ │个人中心│  │
│  └────────┘ └────────┘ └────────┘  │
└─────────────────────────────────────┘
              ↓
┌─────────────────────────────────────┐
│         鸿鹄大模型 API              │
│      OpenAI兼容接口: honghu         │
└─────────────────────────────────────┘
```

## 【技术选型】

| 技术 | 版本 |
|------|------|
| Vue 3 | 3.4+ |
| Vant | 4.8+ |
| Vite | 5.0+ |
| 鸿鹄模型 | honghu |

## 【功能模块】

| 模块 | 文件 | 说明 |
|------|------|------|
| 对话界面 | views/Home.vue | 智能对话+快捷入口 |
| 历史记录 | views/History.vue | 对话历史 |
| 个人中心 | views/Profile.vue | 工单/反馈 |
| API封装 | api/honghu.js | 鸿鹄大模型调用 |

## 【运行方式】

```bash
cd ~/.openclaw/shared-workspace/projects/ai-customer-service/h5
npm install && npm run dev
```
