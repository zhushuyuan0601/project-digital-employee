## 一、联通助理产品分析

| 项目 | 内容 |
|------|------|

| **定位** | 智能语音助手、生活服务助手 |

### 1.2 核心功能

| 功能模块 | 描述 |

## 二、安全管家产品分析

### 2.1 产品定位

| 项目 | 内容 |
|------|------|
| **产品名称** | 安全管家 |
| **定位** | 手机安全防护、骚扰拦截 |
| **目标用户** | 联通手机用户 |
| **核心场景** | 骚扰电话拦截、诈骗防护、隐私保护 |

### 2.2 核心功能

| 功能模块 | 描述 |
|---------|------|
| **骚扰拦截** | 骚扰电话、垃圾短信拦截 |
| **诈骗防护** | 诈骗电话识别、风险提醒 |
| **隐私保护** | 隐私号码、防泄露保护 |
| **安全检测** | 手机安全扫描、病毒查杀 |

### 2.3 用户场景

```
场景1：骚扰电话拦截
来电识别为骚扰电话 → 自动拦截 → 用户收到拦截通知

场景2：诈骗防护
收到可疑短信 → 识别为诈骗 → 风险提醒用户

场景3：隐私保护
外卖/快递下单 → 使用隐私号码 → 保护真实号码
```

## 五、技术架构建议

### 5.1 整体架构

```
┌──────────────────────────────────────────────┐
│                 用户端                        │
│   H5页面 / 小程序 / APP / Web                 │
└──────────────────────────────────────────────┘
                    ↓
┌──────────────────────────────────────────────┐
│              API 网关                         │
│   认证 / 限流 / 路由                          │
└──────────────────────────────────────────────┘
                    ↓
┌──────────────────────────────────────────────┐
│           AI 智能客服核心                     │
│   ├─ NLU 意图识别                            │
│   ├─ 对话管理                                │
│   ├─ 知识库检索                              │
│   └─ 大模型推理（鸿鹄）                      │
└──────────────────────────────────────────────┘
                    ↓
┌──────────────────────────────────────────────┐
│              业务系统                         │
│   CRM / 订单 / 账单 / 工单                   │
└──────────────────────────────────────────────┘
```

### 5.2 技术栈

| 层级 | 技术 |
|------|------|
| **前端** | Vue3 + Vant UI（H5） |
| **后端** | Node.js / Python |
| **大模型** | 鸿鹄大模型（OpenAI 兼容接口） |
| **数据库** | MySQL + Redis + Elasticsearch |
