# 联通智铃 H5 - AI视频彩铃升级版

> 对标电信天翼智铃，实现AI智能体对话引导创作

## 🚀 快速启动

```bash
cd ~/.openclaw/shared-workspace/projects/unicom-caller-ringtone/h5
npm install
npm run dev
```

访问：http://localhost:5176

## ✨ 核心功能

### 1. AI智能体对话引导（对标星小辰）
- 多轮对话引导创作
- 意图识别与需求细化
- 一键进入创作流程

### 2. 创作类型
- 🎊 拜年祝福
- 🏢 企业宣传
- 🎮 IP联动（200+IP库）
- ✨ 自由创作

### 3. 模板库
- 热门模板
- 节日模板
- 企业模板
- IP联动模板

### 4. IP库
- 动漫IP：小猪佩奇、奥特曼、熊出没
- 游戏IP：王者荣耀、原神、英雄联盟
- 影视IP：功夫熊猫、变形金刚
- 体育IP：CBA联赛

### 5. 创作进度
- 实时进度展示
- 预览窗口
- 一键设为彩铃
