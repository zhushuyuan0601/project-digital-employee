# 联通智铃 H5 Demo - 技术架构

> **日期**: 2026-03-09
> **技术负责**: 小开

## 【技术架构】

```
┌─────────────────────────────────────┐
│       Vue 3 + Vant UI (H5)          │
│  ┌────────┐ ┌────────┐ ┌────────┐  │
│  │对话创作│ │快捷入口│ │历史记录│  │
│  └────────┘ └────────┘ └────────┘  │
└─────────────────────────────────────┘
              ↓
┌─────────────────────────────────────┐
│        鸿鹄大模型 (honghu)          │
│       OpenAI兼容API接口             │
└─────────────────────────────────────┘
```

## 【技术选型】

| 技术 | 版本 |
|------|------|
| Vue 3 | 3.4+ |
| Vant | 4.8+ |
| Vite | 5.0+ |
| 鸿鹄模型 | honghu |

## 【核心功能】

| 功能 | 说明 |
|------|------|
| 对话创作 | AI引导生成视频创意 |
| 快捷入口 | 拜年/企业/IP模板 |
| 历史记录 | 本地存储 |

## 【运行方式】

```bash
npm install && npm run dev
```
