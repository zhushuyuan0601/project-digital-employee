## 二、技术选型

### 2.1 技术栈总览

| 层级 | 技术选型 | 版本 | 说明 |
|------|---------|------|------|
| **前端(H5)** | Vue 3 + Vant | 3.4+ | 移动端优先 |
| **前端(小程序)** | uni-app | 3.0+ | 跨平台复用 |
| **后端框架** | FastAPI | 0.110+ | 高性能异步 |
| **智能体框架** | LangChain + LangGraph | 0.1+ | 流程编排 |
| **大模型** | 鸿鹄(GLM-4) | - | 主模型 |
| **备选模型** | 通义千问 / 文心一言 | - | 降级备用 |
| **数据库** | PostgreSQL | 15+ | 业务数据 |
| **缓存** | Redis | 7+ | 会话状态 |
| **向量库** | Milvus | 2.3+ | 语义检索 |
| **消息队列** | RabbitMQ | 3.12+ | 异步任务 |
| **对象存储** | 联通云OSS | - | 媒体文件 |
| **容器化** | Docker + K8s | - | 部署运维 |

### 2.2 大模型选型

| 模型 | 用途 | 优势 | 备注 |
|------|------|------|------|
| **鸿鹄(GLM-4)** | 主对话模型 | 自研、合规、成本低 | 首选 |
| 通义千问 | 备用模型 | 性能稳定 | 降级使用 |
| 文心一言 | 备用模型 | 中文优化 | 降级使用 |

## 三、开发里程碑

### 3.1 总体计划（6周）

```
Week 1-2: Phase 1 - 智能体核心
├── 对话引擎开发
├── 意图识别模块
└── 状态机实现

Week 3-4: Phase 2 - 能力集成
├── 模板推荐引擎
├── 内容生成集成
└── 进度反馈系统

Week 5-6: Phase 3 - 原型完成
├── H5前端开发
├── 集成测试
└── 内测版本发布
```

### 3.2 详细里程碑

| 阶段 | 时间 | 交付物 | 验收标准 |
|------|------|--------|---------|
| **M1** | Week 1 | 智能体对话引擎Demo | 可进行多轮对话 |
| **M2** | Week 2 | 意图识别+状态机 | 意图识别准确率>85% |
| **M3** | Week 3 | 模板推荐引擎 | 推荐命中率>80% |
| **M4** | Week 4 | 内容生成+进度反馈 | 完整创作流程 |
| **M5** | Week 5 | H5原型完成 | 可演示完整流程 |
| **M6** | Week 6 | 内测版本发布 | 通过验收测试 |
