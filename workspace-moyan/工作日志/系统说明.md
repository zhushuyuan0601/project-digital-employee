# 墨言周报生成器

## 系统架构

### 目录结构

```
workspace-moyan/
├── 工作日志/           # 工作日志存储
│   ├── README.md       # 使用说明
│   ├── 周报模板.md      # 周报结构模板
│   ├── 2026-W10.md     # 第 10 周日志
│   └── ...
├── 周报/               # 已生成的周报
│   └── 2026-03-05-墨言 - 周报.md
└── USER.md             # 用户风格配置
```

### 日志格式

用户输入：
```
add：[方向] 工作内容
```

**方向映射**:
- `[核心技术]` → 一、核心技术攻关&&揭榜项目
- `[号码库]` → 二、智能号码库模型研发
- `[代接]` → 三、智能代接能力升级
- `[评测]` → 四、评测能力建设
- `[业务]` → 五、业务能力支撑
- `[其他]` → 六、其他工作

### 周报生成流程

1. **读取本周日志** - 从 `工作日志/2026-W10.md` 读取
2. **分类整理** - 按任务方向分组
3. **生成周报** - 按模板格式输出
4. **保存周报** - 写入 `周报/2026-03-05-墨言 - 周报.md`

### 智能体协作

墨言可以与其他智能体通信，获取相关技术工作内容：
- `lanji` - 知识库管家，查询历史问题和方案
- `main` - 总控路由
- 其他专业 Agent
