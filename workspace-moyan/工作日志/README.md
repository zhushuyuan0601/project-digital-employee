# 工作日志系统

## 使用说明

### 记录工作日志

通过以下方式记录每日工作：

```
add：[任务方向] 具体工作内容
```

**任务方向对应关系**:
- `add：[核心技术] xxx` → 核心技术攻关&&揭榜项目
- `add：[号码库] xxx` → 智能号码库模型研发
- `add：[代接] xxx` → 智能代接能力升级
- `add：[评测] xxx` → 评测能力建设
- `add：[业务] xxx` → 业务能力支撑
- `add：[其他] xxx` → 其他工作

**示例**:
```
add：[核心技术] 完成了 OpenClaw 多智能体配置，包括角色定义和路由机制
add：[评测] 调试了 agent 间通信，解决了超时问题
add：[业务] 学习了 AI Agent 协同工作流程
```

### 日志存储

日志按日期存储，格式为 `YYYY-MM-DD.md`，每周一个汇总文件。

### 周报生成

每周五，墨言会根据本周所有日志自动生成周报。
