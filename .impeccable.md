## Design Context

### Users
- **Target**: Multi-agent system operators and developers
- **Use Case**: Real-time monitoring and coordination of AI agents (CEO, researcher, PM, tech lead, QA)
- **Context**: Professional environment, both bright office and late-night sessions

### Brand Personality
- **Voice**: Technical precision with human warmth
- **3 Words**: Refined · Modern · Trustworthy
- **Emotional Goal**: Confidence in system reliability, clarity in complex information

### Aesthetic Direction
- **Visual Tone**: Super-ellipse geometry (macOS Big Sur inspired), modern tech aesthetic
- **Theme**: Auto-switching light/dark via `prefers-color-scheme`
- **Color Palette**: Deep indigo (#2D3561) + Coral orange (#FF6B6B) + Cream white (#FFF9F0)
- **Anti-references**: Generic AI admin panels, glassmorphism overload, neon-on-dark clichés

### Design Principles
1. **Super-ellipse shapes**: Use `border-radius: 16-24px` for cards, `8-12px` for interactive elements
2. **Perceptual uniformity**: OKLCH color space for consistent contrast across themes
3. **Adaptive depth**: Subtle shadows in light mode, glow effects in dark mode
4. **Typographic clarity**: Fixed `rem` scale for app UI (no fluid type), 1.25 ratio between steps
5. **Motion with purpose**: Deceleration easing only, no bounce/elastic, reduced-motion support
